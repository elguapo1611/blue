# Blue

Blue helps you manage your Ruby on Rails deployment.

* Deploy new code
* Install/Update/Restart services

## Installation

Add this line to your application's Gemfile:

    group :deployment do
      gem 'blue'
    end

And then execute which will move basic config files into place.

    $ rake blue:setup
